Tutorial

Primeiros passos

1. Configure o Supabase

Crie as tabelas com o SQL em supabase/schema.sql e informe aqui a URL e a anon key.

2. Crie sua conta

Use o bloco de autenticação para cadastro/login e começar a escrever dados isolados por usuário.

3. Vincule o Telegram

Gere um código no dashboard e clique em Abrir no Telegram. O bot recebe o código no /start e vincula seu chat automaticamente.

4. Alimente os dados

Use o bot ou o próprio Supabase para registrar transações, ativos, passivos, metas e assinaturas.
