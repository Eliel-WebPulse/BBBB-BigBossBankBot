Finn Finance Dashboard

Conexão opcional com Supabase

Para manter a página estática na Vercel, use apenas a URL do projeto e a chave pública anon. Não coloque chave service role neste HTML.

O dashboard espera colunas compatíveis com seu bot atual: tipo, valor, categoria, descricao e data. As configurações ficam salvas apenas no navegador.
